
APP que simula funcionalidades do PicPay para transações financeiras + Jetpack com Kotlin

O app segue a arquitetura MVVM e apresenta as seguintes tecnologias:

    Retrofit2;
        - Cosumo de Apis Rest
        - Autenticação com JWT Token
    ViewModel;
    LiveData;
    Courotines;
    Koin;
    Navigation;
    Room.
